# 资本资产定价模型 CAPM

## 一、教学基本信息

- **课程名称**：投资学
- **授课章节**：第 5 章 资本资产定价模型 (Capital Asset Pricing Model, CAPM)
- **授课学时**：3 学时（135 分钟）
- **授课对象**：金融学大三本科生
- **前置知识**：均值-方差分析、有效前沿、无差异曲线

## 二、教学目标

| 维度 | 目标 |
|------|------|
| 知识目标 | 理解 CAPM 的核心假设、推导逻辑和最终公式；掌握 β 系数的含义与计算 |
| 能力目标 | 能够用 CAPM 计算资产的预期收益率；能够解读和运用 β 系数进行简单投资分析 |
| 素养目标 | 理解理论假设与现实市场之间的差距，初步培养对金融模型适用性的批判性思维 |

## 三、教学重难点

- **重点**：CAPM 公式的推导与含义、β 系数的经济含义、证券市场线 SML
- **难点**：分离定理与市场组合的概念、β 系数为什么衡量的是系统性风险而非总风险
- **难点突破策略**：先讲直觉（图形），再讲推导（代数），最后回到直觉（经济解释）；用 Excel 现场演示不同 β 值对应的收益特征

## 四、教学过程

### 1. 导入（约 10 分钟）

- **导入方式**：复习衔接 + 问题引入
- 回顾上次课的结论：有效前沿上的组合都是最优的吗？不同投资者的最优风险组合一样吗？
- 引出核心问题：如果市场上所有投资者都按马科维茨方法做投资，市场均衡时资产价格会怎样？

### 2. 新课讲授（约 100 分钟）

| 环节 | 内容要点 | 时间 | 教学方法 |
|------|---------|------|---------|
| 2.1 CAPM 的核心假设 | 8 条假设及其目的（简化现实以得到简洁结论） | 10min | 讲授 + 提问 |
| 2.2 分离定理 | 无风险资产引入后，最优风险组合与投资者偏好分离 | 15min | 图示讲解 |
| 2.3 市场组合与资本市场线 CML | 市场组合的定义、CML 的含义和方程 | 15min | PPT 图示 |
| 2.4 β 系数的引入 | 单个资产对市场组合风险的边际贡献 | 20min | 公式推导 + 直观解释 |
| 2.5 CAPM 公式推导 | E(Ri) = Rf + βi [E(Rm) - Rf] | 15min | 逐步推导 |
| 2.6 证券市场线 SML | SML vs CML 的区别、均衡与非均衡的含义 | 15min | 对比讲解 |
| 2.7 CAPM 的应用与局限 | 实际应用场景、模型局限、后续发展（Fama-French 等） | 10min | 讨论 |

### 3. 案例分析（约 15 分钟）

- **案例名称**：用上证 50 成分股数据估算 β 系数
- **关键问题**：
  - 用近 3 年数据估算的 β 和近 1 年数据估算的 β 差异大吗？
  - β 稳定的股票有哪些特征？
- **讨论要点**：β 的时变性意味着什么？对投资实践的启示

### 4. 课堂小结（约 10 分钟）

- CAPM 从马科维茨走向均衡的关键一步：引入无风险资产 + 同质预期
- β 是唯一被定价的风险——因为它无法通过分散化消除
- CML 描述有效组合、SML 描述所有资产（包括无效组合和单个资产）
- CAPM 的局限：假设太强、单因子、实证表现不佳——这也是后续因子模型发展的动因

## 五、板书设计 / PPT 结构

```
有效前沿 → +无风险资产 → CML → +均衡 → CAPM
         → 分离定理       → 市场组合
                          → β 系数 → SML → 应用
```

## 六、课后任务

- **复习要点**：CAPM 假设 → 分离定理 → CML → β → SML 的逻辑链
- **课后练习**：教材课后习题第 1、3、5、8 题
- **预习内容**：第 6 章 指数模型与套利定价理论 APT
